‚úÖ Method 1: Right-Click Override (Easiest)

  1. Right-click (or Control + Click) on Sparcle.app in your Applications folder
  2. Select "Open" from the menu
  3. Click "Open" in the confirmation dialog

‚úÖ Method 2: System Settings (Alternative)

  1. Try to open Sparcle.app (it will be blocked)
  2. Open System Settings ‚Üí Privacy & Security
  3. Scroll down to find: "Sparcle was blocked from opening"
  4. Click "Open Anyway"
  5. Confirm by clicking "Open" in the dialog

Why does this happen?

Alpha builds are not notarized by Apple (which requires a $99/year developer account). This security override is a one-time step - after you approve it once, Sparcle will open normally.

System Requirements

  • macOS 13.0 (Ventura) or later
  • Apple Silicon (M1/M2/M3) or Intel processor
  • Minimum 4GB RAM (8GB recommended)
  • 100MB available disk space
  • Internet connection for AI features

Known Issues (Alpha)

This is an alpha release - expect bugs and rough edges. Known issues:

  • First launch may take ~10 seconds to initialize
  • Some UI animations may be janky
  • Crash reporting not yet implemented
  • No auto-update mechanism yet (manual downloads for updates)
